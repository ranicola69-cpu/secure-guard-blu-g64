## How to Contribute

### 1. Fork the Repository
```bash
git clone https://github.com/YOUR_USERNAME/secure-guard-blu-g64.git
cd secure-guard-blu-g64
```

### 2. Create a Feature Branch
```bash
git checkout -b feature/amazing-feature
```

### 3. Make Your Changes
- Follow existing code style
- Write clear commit messages
- Add comments for complex logic
- Test your changes thoroughly

### 4. Commit Your Changes
```bash
git add .
git commit -m 'feat: Add amazing feature'
```

### 5. Push to Your Fork
```bash
git push origin feature/amazing-feature
```

### 6. Open a Pull Request
- Go to the original repository
- Click "New Pull Request"
- Select your branch
- Describe your changes in detail

## Code Style Guidelines

### Backend (Python/FastAPI)
- Follow PEP 8 style guide
- Use type hints
- Add docstrings for functions
- Keep functions small and focused

### Frontend (React Native/TypeScript)
- Use TypeScript for type safety
- Follow React hooks best practices
- Use functional components
- Keep components under 300 lines

## Commit Message Format

Use conventional commits:
```
feat: Add new feature
fix: Fix bug in security scanner
docs: Update README
style: Format code
refactor: Refactor app manager
test: Add tests for VPN module
chore: Update dependencies
```

## Reporting Bugs

When reporting bugs, please include:

1. **Device Information**
   - Device model (e.g., Blu G64)
   - Android version
   - Shizuku version

2. **Steps to Reproduce**
   - Clear step-by-step instructions
   - Expected behavior
   - Actual behavior

3. **Logs**
   - Error messages
   - Crash logs
   - Screenshots if applicable

## Feature Requests

When requesting features:

1. Check if feature already exists
2. Search existing issues
3. Clearly describe the feature
4. Explain the use case
5. Provide examples if possible

## Testing

Before submitting:

### Backend Testing
```bash
cd backend
pytest tests/
```

### Frontend Testing
```bash
cd frontend
yarn test
```

### Manual Testing
- Test on physical device
- Verify Shizuku integration
- Check all screens and features
